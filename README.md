# CAV Concept

## Русский

Это публикация концепта CAV: Context, AI, Viewer.

CAV — это архитектура AI-среды, в которой Context хранит и накапливает рабочую реальность, AI преобразует её, а Viewer выступает отдельным слоем фокуса. Результат работы не исчезает как разовый ответ, а возвращается в Context и становится материалом следующего шага.

Репозиторий открыт как публичная фиксация текущей формулировки концепта. Основной текст находится в файле `Contex-Ai-Viewer.md`.

## English

This repository presents the CAV concept: Context, AI, Viewer.

CAV is an AI environment architecture in which Context stores and accumulates the working reality, AI transforms it, and Viewer acts as a separate focus layer. The result does not disappear as a one-off response, but returns to Context as material for the next step.

The repository is intended as a public record of the current formulation of the concept. The main text is available in `Contex-Ai-Viewer.md`.
